这个写法实在是有些别扭，而且非常不利于编写，并且还和其他的unix系列系统不一致，所以解决这个问题的终极解决办法是：

切换成gnu_sed

1.brew install gnu-sed –with-default-names

2.vi ~/.zshrc export PATH=“/usr/local/opt/gnu-sed/libexec/gnubin:$PATH”

3.source ~/.zshrc 或者新开窗口，让设置生效
